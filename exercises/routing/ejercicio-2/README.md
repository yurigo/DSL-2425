# Ejercicio 2

Crea una API REST donde podremos realizar una petición por cada número del 1 al 99 y devuelva la tabla de multiplicar del número correspondiente.

## Request
```
GET /9
```

## Response
```
{
    "1x9" : 9,
    "2x9" : 18,
    "3x9" : 27,
    "4x9" : 36,
    ...
}
```
